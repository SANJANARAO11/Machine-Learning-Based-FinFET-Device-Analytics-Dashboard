Project Overview

This project presents a TCAD-inspired FinFET Device Analytics Dashboard developed using Python, Machine Learning, and Power BI.
The objective is to analyze and predict semiconductor device performance metrics such as Drain Current (Id), Delay, and Power using Random Forest Regression models.

Technologies Used

Python, Numpy, Pandas, Scikit-Learn, Power BI
Dataset Features
Gate Length (Lg), Fin Width (Wfin), Fin Height (Hfin), Oxide Thickness (Tox), Gate Voltage (Vgs), Drain Voltage (Vds), Doping Concentration, Predicted Outputs, Drain Current (Id), Delay, Power

Overview of key device performance metrics.

Device Parameter Analysis
Analysis of the impact of physical and electrical parameters.
Machine Learning Model Validation

Key Findings

Delay increases with Gate Length.
Fin Width significantly influences Drain Current.
Oxide Thickness affects Power Consumption.
Machine Learning models successfully predicted device performance metrics.
Clear power-delay trade-off behavior was observed.
